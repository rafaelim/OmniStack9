# OmniStack 9.0

My implementation for the AirCnC of the omnistack week 9 from Rocketseat.

## Getting Started
### Prerequisites
- Yarn
- Lerna
- docker-compose
- docker

### Installing
`lerna bootstrap`

## Running
### Api
`docker-compose up -d`
### App
`yarn start`
### Mobile
`yarn start`

## Stack
### API 
- NodeJS 
- MongoDB

### App
- ReactJS

### Mobile
- React-Native
